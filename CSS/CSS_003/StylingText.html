Text and Font Styling

Fundamentals of text/font styling. Example font weight, family and style, text alignment,line and letter spacing

The CSS properties used to style text generally fall into two categories

  1. Font Style
  2. Text layout

  • Font Styles

    Properies that affect a text's font , e. g. which font gets applied, it's size, bold, italic, etc

  • Text Layout Styles

    Properties that affect the spacing and other layout features of text such as space between lines and letters, text alignment
